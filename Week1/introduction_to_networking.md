# Introduction to Computer Networks

---

## 1. What is a Computer Network?

A **computer network** is a system where **two or more devices are connected together** so that they can **communicate and share data**.

In a network, devices exchange information using **wired connections (cables)** or **wireless signals (Wi-Fi)**.

The main purpose of a network is to allow devices to **send and receive data efficiently**.

### Simple Example

Two computers connected together form a basic network.
    Computer A -------- Computer B


Computer A can send data to Computer B, and Computer B can send data back.

---

# 2. Why Computer Networks Are Needed

Before networking existed, sharing files between computers required **physical storage devices** such as:

- USB drives  
- CDs  
- External hard drives  

This process was slow and inconvenient.

Computer networks solve this problem by allowing **direct communication between devices**.

---

## Main Benefits of Networking

### 1. File Sharing

Users can easily share files between computers connected to the same network.

Example:
- Students accessing shared notes in a computer lab.

---

### 2. Resource Sharing

Multiple computers can share hardware resources such as:

- Printers  
- Storage systems  
- Internet connections  

Example:
- Several computers in an office using one printer.

---

### 3. Communication

Networks enable different types of communication:

- Email  
- Messaging applications  
- Video conferencing  
- Online collaboration tools  

---

### 4. Internet Access

Networks allow multiple devices to access the **internet using a single connection**.

Example:
- In a home network, a router allows phones, laptops, and smart TVs to share the same internet connection.

---

# 3. Examples of Computer Networks

## Home Network

A typical home network may include:

- Smartphones  
- Laptops

All these devices connect through a **Wi-Fi router**.

This allows them to:
- Access the internet

## The Internet

The **Internet** is the **largest computer network in the world**.

It is made up of **millions of smaller networks connected together globally**.

When you open a website, your computer communicates with a **server** somewhere on the internet.

---

# 4. Basic Components of a Computer Network

A computer network consists of several components that allow communication to occur.

The two main categories are:

- **End Devices**
- **Network Devices**

---

## 4.1 End Devices

End devices are the devices that **send or receive data** in a network.

They are called **end devices** because they exist at the **edges of the network**, where communication starts or ends.

### Examples of End Devices

- Desktop computers  
- Laptops  
- Smartphones  
- Printers  
- Tablets  
- Servers  

### Example Scenario

When you open a website:

1. Your **laptop sends a request**.
2. A **web server sends the webpage back**.

Both the laptop and the server are **end devices**.

---

## 4.2 Network Devices

Network devices help **connect end devices and transfer data between them**.

They form the **infrastructure of the network**.

### Examples of Network Devices

- Switches  
- Routers  
- Wireless access points  
- Modems  

These devices ensure that data moves from the **source device** to the **destination device**.

---

# 5. Simple Network Example

A very basic network structure looks like this:
    Computer A ---- Network Device ---- Computer B


### In this example:

- **Computer A** sends data
- The **network device** forwards the data
- **Computer B** receives the data

This simple concept forms the basis of all larger networks.

---

# 6. Key Concepts Learned Today

From today’s lesson:

- A **computer network** connects devices so they can communicate and share data.
- Networks provide benefits such as:
  - File sharing
  - Resource sharing
  - Communication
  - Internet access
- Devices in a network are classified into:
  - **End devices** – devices that send and receive data
  - **Network devices** – devices that help transfer data between devices
- The **Internet** is the largest network and connects millions of smaller networks worldwide.
